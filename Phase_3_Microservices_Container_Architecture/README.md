# Phase 3 — Modern Microservices & Container Architectures

## Overview

Phase 3 transitions the lab from a manually-configured, single-service environment into a modern containerised infrastructure built on top of the hardened Ubuntu Server (`LNX-SRV-01`) from Phase 2.

## Sub-Phases

| Phase | Title | Status |
|-------|-------|--------|
| 3.1 | Docker Installation | 
| 3.2 | Portainer Container Deployment | 
| 3.3 | Data Lifecycle Engineering | 
| 3.4 | Nginx Reverse Proxy | 
| 3.5 | Ansible Configuration Management |

## Environment
- **Host:** `LNX-SRV-01` — Ubuntu Server 26.04 LTS, `10.0.0.20`
- **Container Runtime:** Docker Engine 29.1.3
- **Containerised Service:** Portainer Community Edition 2.39.4 LTS
